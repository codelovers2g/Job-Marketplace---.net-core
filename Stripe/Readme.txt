Stripe Integration
To send and receive payments from clients to providers for providing services.

- Create Customers (customer who will make payments)
- Attach Credit/Debit Card
- Set Default Card / Delete Cards
- Create custom connect accounts (For receiving payments) 
- Attach Debit card with connect account
- Charge (Destination Charge)
- Payout (Mannual)
- Refund 

Stripe Webhook
To receive updates from Stripe and update payment and account information in application
